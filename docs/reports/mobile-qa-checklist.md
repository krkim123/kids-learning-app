# 모바일 QA 체크리스트 (실기기 기준)

## 목적
- 화면 겹침/잘림 없이 주요 플로우가 동작하는지 확인
- 버튼 간격/터치 영역이 충분한지 확인
- 초기 로딩 체감 속도와 캐시 갱신 동작 확인

## 테스트 기기/해상도
- 360x800 (Android 소형)
- 390x844 (iPhone 12/13/14)
- 412x915 (Android 대형)
- 768x1024 (태블릿 세로)

## 핵심 플로우
1. 앱 첫 진입
2. 프로필 선택/생성
3. 홈 > 추천 카드 > 학습 화면
4. 홈 > 놀이 > 숫자 세기/색칠하기
5. 보상함/출석/부모 페이지
6. 새로고침 후 상태 유지 확인

## 레이아웃 체크
- [ ] 탭바 아이콘/라벨 겹침 없음
- [ ] 홈 빠른놀이 카드 줄바꿈 시 카드 겹침 없음
- [ ] 단계 카드(학습/게임) 텍스트 잘림 없음
- [ ] 색칠하기 팔레트/툴 버튼 줄바꿈 시 버튼 겹침 없음
- [ ] 숫자세기 화면 이모지/선택지 겹침 없음
- [ ] 세로 스크롤 영역에서 가로 스크롤 불필요

## 터치/접근성 체크
- [ ] 주요 버튼(뒤로/기본/보조/툴) 누르기 쉬움
- [ ] 최소 터치 영역 44px 이상
- [ ] 버튼 연타 시 오동작 없음
- [ ] 길게 눌러도 의도치 않은 확대/선택 없음

## 성능 체크
- [ ] 첫 진입 시 흰 화면 대기 과도하지 않음
- [ ] 새로고침 후 캐시 갱신으로 최신 UI 반영
- [ ] 홈/놀이 전환 시 체감 지연 없음
- [ ] 색칠하기 진입 후 팔레트 렌더링 지연 없음

## 캐시/배포 체크
- [ ] 서비스워커 버전 변경 후 1~2회 새로고침으로 최신 리소스 반영
- [ ] 오프라인/저속 네트워크에서 기본 화면 폴백 동작

## 회귀 체크 (이번 변경 기준)
- [ ] UI 텍스트가 한글로만 표시
- [ ] 프로필 이름 비어도 헤더 빈칸/깨짐 없음
- [ ] 색칠하기에서 API 키 입력 UI가 보이지 않음
- [ ] 숫자세기 장면이 매 문제 다양하게 바뀜
