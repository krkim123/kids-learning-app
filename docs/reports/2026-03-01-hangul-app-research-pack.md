# 2026-03-01 한글 학습 앱 벤치마크/커리큘럼/라이선스 리서치

## 적용 원칙
- 경쟁 앱의 UI/캐릭터/이미지를 그대로 복제하지 않는다.
- 기능 패턴(스트릭, 짧은 세션, 필순 연습, 부모 리포트, SRS)만 차용한다.
- 이미지/음원은 라이선스 허용 소스만 사용한다.

## 벤치마크 핵심
- 비교 앱 10개(국내+글로벌): 핑크퐁, 링고애니, 소중한글, Dinolingo, Gus, Droplets, Drops Korean, LingoDeer, Duolingo, Write It Korean
- 반복 패턴:
  - 짧은 세션(5~15분)
  - 스트릭/리마인더
  - 필순/쓰기 강화
  - 부모 대시보드
  - 수집형 보상

## 커리큘럼 핵심
- 학습 단계: 구어/어휘 -> 음운 인식 -> 자모-음가 -> 해독 확장 -> 유창 읽기 -> 쓰기/철자 -> 이해/표현
- 권장 페이싱:
  - 4~5세: 주 4회, 15~20분
  - 6~7세: 주 5회, 25~35분
  - 8~9세: 주 5회, 35~45분

## 라이선스 안전 리소스(우선)
- Heroicons (MIT)
- Lucide (ISC)
- Bootstrap Icons (MIT)
- Material Symbols (Apache-2.0)
- Kenney (CC0)
- Pexels
- Mixkit
- Wikimedia Hangeul Stroke Order (파일별 라이선스 확인)

## 코드 반영 위치
- js/benchmark.js: 앱 비교 데이터, 문해 단계, 연령별 페이싱, 라이선스 리소스 섹션
- css/style.css: benchmark-link-note 스타일
