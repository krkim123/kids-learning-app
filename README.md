# 도경소빈의 요정 교실 🧚‍♀️

6살 도경이와 3살 소빈이를 위한 한글/숫자/영어 학습 앱입니다.

## 사용법

### 스마트폰에서 사용하기

1. 스마트폰 브라우저(크롬/사파리)로 앱 주소에 접속합니다
2. **홈 화면에 추가**를 눌러 앱처럼 설치합니다
   - **안드로이드**: 크롬 메뉴(⋮) → "홈 화면에 추가"
   - **아이폰**: 사파리 공유(↑) → "홈 화면에 추가"
3. 홈 화면의 아이콘을 눌러 실행하면 됩니다

### PC에서 사용하기

`index.html` 파일을 더블클릭하면 브라우저에서 바로 실행됩니다.

## 기능

- **프로필 선택**: 도경(공주님)과 소빈(꽃요정) 각각의 난이도에 맞춤
- **학습 카드**: 한글 자음/모음, 숫자 1~10, 영어 A~Z
- **게임 3종**: 글자 맞추기 퀴즈, 짝 맞추기, 소리 듣고 찾기
- **보상 시스템**: 별, 스티커 30종, 배지 13종 수집
- **음성 재생**: 터치하면 발음을 들려줍니다

## 연령별 차이

| 구분 | 소빈 (3세) | 도경 (6세) |
|------|-----------|-----------|
| 퀴즈 | 2지선다 | 4지선다 |
| 매칭 | 4장 (2쌍) | 8장 (4쌍) |
| 글자 크기 | 크게 | 보통 |
| 힌트 | 3초 후 자동 | 수동 |
| 오답 | 즉시 정답 표시 | 재시도 기회 |

## 오프라인 지원

인터넷 없이도 사용 가능합니다 (최초 1회 접속 후).
